## Tech Stack

- React
- Vite
- CSS/SCSS
- JavaScript/TypeScript

## Getting Started

1. Clone the repository
```bash
git clone [repository-url]
```

2. Install dependencies
```bash
npm install
```

3. Run development server
```bash
npm run dev
```
